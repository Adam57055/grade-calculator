# Overview
This software helps determine needed grades for a desired final grade. For instance, a user may be anxious about a test result and would want
to see what the worst grade would do to her overall grade.

# Background
As an Anatomy student, I would run into a couple of exams where I was worried about my performance and wanted to know what the worst
result would do to my grade. To combat this problem, I wanted to create a software that solved this simply.
